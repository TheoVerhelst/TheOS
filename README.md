# TheOS

TheOS is an operating system written in C++ for learning purposes, forked from
BermudOS.

# Why ?
See https://github.com/RobinPetit/BermudOS for rationale.

# Requirements
 * CMake 3.2 or higher
 * i686-elf-g++ 6.2
 * nasm

# Installation & run

    mkdir build && cd build
	cmake ..
	make
	./run.sh
